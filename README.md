## Comandos de actualización del git:

```bash
#Añade todos los ficheros a git
git add -A
#Añade el siguiente mensaje como descripción del cambio
git commit -m "Mensaje del commit"
#Sube los cambios a la nube
git push
```

## Bajar cambios de la nube:

```bash
git pull
```
